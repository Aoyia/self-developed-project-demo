启动：

## 1. 安装依赖

首先，在项目根目录下运行以下命令来安装所需依赖：

```bash
pnpm install
# npm i
```

## 2. 打包自开发文件

接下来，使用以下命令来打包你开发的文件：

```bash
pnpm build
# npm run build
```

## 3. 启动 Demo

最后，按照以下步骤启动 Demo：

- **安装浏览器插件**：[Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer)  
  VSCode 中安装 `Live Server` 插件
  
- **启动 Demo 页面**：  
  找到 `demo.html` 文件，右键选择 `Open with Live Server` 启动 Demo
